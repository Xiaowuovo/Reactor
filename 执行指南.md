# 🎯 项目重组执行指南

## 📋 执行步骤（仅需3步）

### 第1步：运行重组脚本

```bash
# Windows (使用Git Bash或WSL)
chmod +x reorganize.sh
./reorganize.sh
```

**脚本会自动：**
- ✅ 创建新目录结构（src/ tests/ tools/ docs/ output/）
- ✅ 移动源代码到 src/
- ✅ 移动测试代码到 tests/
- ✅ 移动工具脚本到 tools/
- ✅ 移动文档到 docs/
- ✅ 删除过时文档
- ✅ 整理输出文件到 output/

### 第2步：更新构建系统

```bash
# 使用新的Makefile
mv Makefile.final Makefile
```

### 第3步：验证和清理

```bash
# 测试编译
make clean
make

# 测试运行
make demo

# 删除临时文件（可选）
rm -f reorganize.sh CLEANUP_GUIDE.md 执行指南.md
```

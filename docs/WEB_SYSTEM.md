# 🌐 Muduo 网络库的重构与内存池的实现 - Web监控系统

## 系统概述

这是一个**完整的前后端分离系统**，包含：
- **后端**：基于Reactor的高性能HTTP服务器
- **前端**：实时性能监控Web界面
- **核心库**：线程局部内存池网络库

## 技术栈

### 后端技术
- **语言**: C++11
- **网络模型**: Reactor (Epoll ET模式)
- **内存管理**: 线程局部内存池
- **并发模型**: 主从Reactor + 线程池
- **协议**: HTTP/1.1
- **数据格式**: JSON

### 前端技术
- **语言**: HTML5 + CSS3 + JavaScript
- **UI框架**: 原生CSS (渐变、动画)
- **通信**: Fetch API (AJAX)
- **可视化**: 内嵌图表展示

## 核心功能模块

### 1. HTTP服务器模块
**文件**: `src/HttpServer.h`, `src/webserver.cpp`

**功能**:
- HTTP请求解析 (HttpRequest)
- HTTP响应生成 (HttpResponse)
- 路由管理 (Router)
- 静态文件服务
- RESTful API

**API接口**:

| 端点 | 方法 | 功能 |
|------|------|------|
| `/` | GET | 主页面 |
| `/api/status` | GET | 获取系统状态 |
| `/api/test/mempool` | POST | 运行内存池测试 |
| `/api/test/network` | POST | 运行网络测试 |
| `/api/data` | GET | 获取测试数据 |

### 2. Web监控界面
**位置**: HTTP响应内嵌HTML

**功能**:
- 实时系统状态显示
- 一键测试控制
- 测试日志查看
- 性能数据展示

**UI组件**:
- Dashboard卡片
- 测试按钮
- 实时日志终端
- 数据统计面板

### 3. 性能测试模块
**文件**: `tests/test_mempool.cpp`, `tests/test_network.cpp`

**测试项目**:
- 单线程性能测试
- 多线程扩展性测试
- 网络QPS测试
- 延迟分布测试

**数据输出**:
- CSV格式数据文件
- 可视化图表 (PNG)

## 性能指标

### HTTP服务器性能
- **请求处理**: 基于Reactor高效处理
- **内存管理**: 使用线程局部内存池
- **并发能力**: 支持多线程SubReactor

### 内存池性能
- **单线程加速**: 3.75x (vs malloc/free)
- **多线程加速**: 5.25x (10线程)
- **内存碎片**: 接近0

### 网络性能
- **QPS**: 52,376 req/s
- **平均延迟**: 13.5 μs
- **P99延迟**: 降低6.67x

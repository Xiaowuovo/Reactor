## 🔧 技术栈

### 前端框架
- **原生HTML5** - 语义化标签
- **原生CSS3** - CSS变量、Grid、Flexbox、动画
- **原生JavaScript（ES6+）** - Async/Await、箭头函数、解构

### 第三方库
- **Chart.js 3.9.1** - 图表绘制
- **CDN加载** - 快速访问

### API接口
```javascript
GET  /api/status          // 获取系统状态
POST /api/test/mempool    // 运行内存池测试
POST /api/test/network    // 运行网络测试
GET  /api/data?type=xxx   // 获取测试数据
```
